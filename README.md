**Twse Monitor** is a free Android application for real-time monitoring of Taiwanese stock market data (TSE and OTC). It allows users to track stock prices, volume, and set alerts for price thresholds or percentage changes. The APK is freely available for download and use.

---

### Quick Features
- Live updates for TSE & OTC stocks  
- Set upper/lower price alerts  
- Add, edit, or remove stocks easily  
- Notifications when thresholds are reached  
- Background monitoring even if app is closed  

---

### How to Use
1. Download the APK from this Release.  
2. Install on your Android device.  
3. Open the app to see your watchlist.  
4. Tap a stock to view real-time data.  
5. Edit alerts or add new stocks by code.
